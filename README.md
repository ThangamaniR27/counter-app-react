# 🔢 Counter App (React)

A simple React Counter Application to increase, decrease, and reset a number.

---

## 🚀 Features

* ➕ Increase count
* ➖ Decrease count
* 🔄 Reset count
* ⚡ Fast and responsive UI
* 🎯 Built using React Hooks

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6)
* HTML5
* CSS

---

## 📂 Project Structure

```bash
src/
 ├── App.js
 └── components/
      └── Counter.js
```

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/counter-app.git
```

2. Navigate to the project

```bash
cd counter-app
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

---

## 🎯 How It Works

* Click **+** to increase the count
* Click **-** to decrease the count
* Click **Reset** to reset the value to 0
